# Финальный проект 1 семестра

REST API сервис для загрузки и выгрузки данных о ценах.

## Требования к системе
 - Go 1.23
 - PostgreSQL 15

## Алгоритм взаимодействия с приложением

1. Выполнение миграций и компиляция приложения
    ```bash
    ./scripts/prepare.sh
    ```

2. Запуск приложения
    ```bash
    ./scripts/run.sh
    ```
   
3. Запуск тестов
   ```bash
   ./scripts/tests.sh 3
    ```
